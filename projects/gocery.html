GoCery

Breeze through the supermarket using crowd-sourced aisle information

  • Website: https://gocery-app.netlify.com/
  • Source code: GitHub
  • Node.js
  • MongoDB
  • React
  • Yelp

GoCery aims to help shoppers stay organized in the grocery store by sorting their lists by aisle. GoCery uses the aisle data that other user's have provided. While you're in the store and checking off items, GoCery displays Waze-like prompts to make it easy for a user to quickly update the aisle information for the product she just found.
